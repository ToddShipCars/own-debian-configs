================================================================================

For changing kde/gnome terminal colors execute and check this:
view_terminal_colors.sh

================================================================================

To be able all plugins for VIM to work you should install gvim:
sudo apt-get install vim-gtk vim-gnome ncurses-term

================================================================================

In terminator config:
~/.config/terminator/config

update this:
palette = "#7e7e7e:#ff6c60:#a8ff60:#ffffb6:#96cbfe:#ff73fd:#c6c5fe:#eeeeee:#9c9c9c:#ffb6b0:#ceffab:#ffffcb:#b5dcfe:#ff9cfe:#dfdffe:#ffffff"
